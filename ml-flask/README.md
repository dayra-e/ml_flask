# ml-flask

Esta es una aplicación flask que carga un modelo previamente entrenado.

## Estructura del proyecto

- `src/app.py`: Contiene el código de la aplicación flask.
- `output/model.pkl`: Contiene el modelo entrenado.
- `data/data.csv`: Contiene los datos de entrenamiento.
- `templates/main.html`: Contiene la página web de la aplicación.
